# ShoppyGlobe Backend API

##  Project Description

This project is a backend API for an e-commerce application built using Node.js, Express.js, and MongoDB.

##  Features

* Product API (Get all products, Get single product)
* Cart API (Add, Update, Delete items)
* JWT Authentication (Register & Login)
* Protected routes for cart
* MongoDB database integration

##  Technologies Used

* Node.js
* Express.js
* MongoDB (MongoDB Compass)
* JWT Authentication
* Mongoose

##  API Endpoints

### Auth

* POST /register
* POST /login

### Products

* GET /products
* GET /products/:id

### Cart (Protected)

* POST /cart
* PUT /cart/:id
* DELETE /cart/:id

##  How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open:
   http://localhost:5000
